## Project Overview
This project analyzes customer churn data to identify patterns and factors that influence
customer retention. The analysis was conducted as part of my MSc Data Analytics coursework,
with a focus on translating analytical results into business-relevant insights.

## Objectives
- Explore customer behavior and churn patterns
- Identify key variables associated with churn
- Support data-driven customer retention strategies

## Tools & Techniques
- Python (pandas, numpy, scikit-learn)
- Exploratory data analysis
- Predictive modeling for churn analysis

## Key Insights
- Identified key features associated with higher churn risk
- Demonstrated how data analysis can support proactive customer success actions

## Business Relevance
The project highlights how analytical insights can be used to improve customer satisfaction
and retention through targeted interventions.

## Context
Academic project completed as part of the MSc Data Analytics program.
